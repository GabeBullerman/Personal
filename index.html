Web App

Lusiant is a webapp created using JavaScript, HTML and CSS. This site is a user-friendly, interactive ecommerce store for a clothing brand created by a childhood friend. This webapp is powered by a MongoDB database, and React for the frontend. Lusiant utilizes both tailwind and bootstrap CSS libraries for icons, and the building blocks of the catalog section of the site. This site also feature an administrator lock to shut down the site between productions of Lusiant products.
